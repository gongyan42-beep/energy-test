# 能量频率层级测评

基于霍金斯能量层级理论的心理测评H5页面。

## 功能特点

- 12道单选题测评
- 基于霍金斯能量层级评分
- 五个能量层级划分
- 可视化图表展示
- 详细解读和提升建议
- 移动端适配

## 技术栈

- HTML5 + CSS3 + JavaScript
- Chart.js（图表）
- Nginx（静态服务）
- Docker（容器化部署）

## 本地运行

```bash
# 使用 Python 简单服务器
python3 -m http.server 5006

# 或使用 live-server
npx live-server --port=5006
```

## Docker 部署

```bash
docker-compose up -d --build
```

## 端口

- 本地开发：5006
- 服务器部署：3006
